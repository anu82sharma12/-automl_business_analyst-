Built with Streamlit + scikit-learn to make predictive analytics accessible to non-technical users.

###  Key Results
- Achieved **30% better interpretability** via visual feature mapping  
- Generates quick **sales forecasts and KPI scenarios**  
- 100% reproducible, runs locally with zero setup
